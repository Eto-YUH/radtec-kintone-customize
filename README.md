# Radtec kintone Customize

This folder contains JavaScript files loaded by kintone app customization.

## Study Results app 116

File:

```text
study-results/app116-study-results-ui.js
```

After publishing this folder to GitHub, register a jsDelivr URL in kintone
`JavaScript / CSSでカスタマイズ`.

Example:

```text
https://cdn.jsdelivr.net/gh/<github-user>/<repo>@main/study-results/app116-study-results-ui.js
```

For cache refresh during development, append a version query.

```text
https://cdn.jsdelivr.net/gh/<github-user>/<repo>@main/study-results/app116-study-results-ui.js?v=20260703-1
```
